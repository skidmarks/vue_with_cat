여러 개의 속성 데이터 바인딩하기

v-bind 디렉티브의 매개변수 부분을 생략하고 객체를 다음과 같이 전달하면 한 번에 바인딩할 수 있으며, 템플릿의 가독성도 굉장히 향상된다.

매개변수로 전달된 v-bind와 함께 특정 요소함을 따로 지정할 수도 있다.

SVG 데이터 바인딩하기

입력 양식 슬라이더의 숫자와 SVG를 동기화해서, 원의 반지름을 변화시킨다.
